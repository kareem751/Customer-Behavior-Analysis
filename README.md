# Customer-Behavior-Analysis
Customer Behavior Analysis Dashboard Using power BI, Python and SQL

A complete data analytics project that analyzes customer shopping behavior using **SQL, Python, and Power BI** to generate actionable insights and interactive dashboards.

---

##Project Overview

This project focuses on understanding customer purchasing patterns using real-world retail data. It includes:

- Data cleaning & transformation (Python)
- Data querying & analysis (SQL)
- Interactive dashboard creation (Power BI)

The goal is to help businesses make **data-driven decisions**.

---

##Project Files

- `customer_shopping_behavior.csv` → Raw dataset  
- `customer_behavior_sql_queries.sql` → SQL queries for analysis  
- `sales.ipynb` → Data preprocessing & analysis in Python  
- `customer_behavior_dashboard.pbix` → Power BI dashboard  

---

##Tech Stack

- **Python** (Pandas, NumPy, Matplotlib)
- **SQL** (PostgreSQL / MySQL)
- **Power BI**
- **Jupyter Notebook**
